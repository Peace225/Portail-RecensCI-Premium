
# RecensCI Advanced Platform

Version avancée de la plateforme RecensCI avec :
- Frontend React + TypeScript + Tailwind PWA
- Backend Node.js + Express + Firebase + PostgreSQL
- Microservices pour ingestion, validation, déduplication
- Dashboards, Heatmaps, Notifications
- Authentification JWT + RBAC
- Observabilité et DevOps (Monitoring + Logs + CI/CD)

Cette structure est prête pour développement et tests.
